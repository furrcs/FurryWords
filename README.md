# 🐾 Furry Words

**Интерактивное приложение для изучения английских слов**

## 📖 О программе
Приложение для изучения английских слов в игровой форме. Показывается слово, нужно выбрать правильный перевод.

## ✨ Возможности
- 🎮 Изучение слов (тесты с вариантами ответов)
- 📊 Статистика прогресса
- ⚙️ Настройки сложности
- 🗄️ Редактор базы данных (добавление/удаление слов)
- 📥 Импорт/экспорт слов из файлов
- 🔍 Поиск в редакторе

## 🚀 Запуск

**Windows:** `python main.py` (или запустите `run.bat`)  
**Linux:** `sudo apt install python3-tk && python3 main.py`

## 📥 Формат файла для импорта
```
слово,перевод1,перевод2
cat,кот,кошка
dog,собака
```
Поддерживаются разделители: `,` `;` `табуляция`

## 📁 Файлы
- `main.py` - программа
- `data.db` - база данных (создаётся автоматически)
- `data.txt` - файл для импорта
